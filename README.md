# VidYa
VidYa is a video calling application 

Wrapper application around the Zego Video Call SDK
https://www.zegocloud.com/product/video-call

## Features
- [ ] User Authentication
- [ ] Group Codes 
- [ ] One to One Calls
- [ ] Group Calls
- [ ] In call chats

## Tech 

- **Backend** - SpringBoot
- **FrontEnd** - React/Angular
- **Databases** - Postresql
- **Deployment** - Docker
